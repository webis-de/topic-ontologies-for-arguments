Setup
```
python3 -m venv myvenv
source myvenv/bin/activate
pip3 install --upgrade pip
pip3 install -r requirements.txt
```

Quickstart
```
python3 src/python/direct-match.py --help
python3 src/python/direct-match.py --documents-file test-documents.csv --topics-file test-topics.csv --output-file test-output.csv
```
